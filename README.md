# 🚦 Road Infrastructure Detection System

Система детекции дорожной инфраструктуры с использованием компьютерного зрения для анализа видео с автомобильных дорог.

## 📁 Структура проекта

```
mashin/
├── src/                          # Исходный код
│   ├── core/                     # Основные модули
│   │   ├── detectors/           # Детекторы объектов
│   │   ├── trackers/            # Трекеры (GPS, пикетаж)
│   │   └── processors/          # Обработчики данных
│   ├── telegram/                # Telegram бот
│   └── utils/                   # Утилиты
├── scripts/                      # Скрипты для запуска
│   ├── detection/               # Скрипты детекции
│   ├── training/                # Скрипты обучения
│   ├── download/                # Скрипты загрузки
│   ├── processing/              # Скрипты обработки
│   └── utils/                   # Утилиты
├── data/                        # Данные
│   ├── models/                  # Обученные модели
│   ├── datasets/                # Наборы данных
│   └── results/                 # Результаты обработки
├── docs/                        # Документация
├── tests/                       # Тесты
└── tools/                       # Инструменты
```

## ⚡ Оптимизации производительности

Для предотвращения перегрева ПК реализованы следующие оптимизации:

### 🎬 Обработка видео
- **Пропуск кадров**: Обрабатывается каждый N-й кадр (зависит от FPS)
- **Ограничение длительности**: Максимум 15 минут видео
- **Ограничение размера**: Максимум 50 MB
- **Масштабирование**: Автоматическое уменьшение разрешения до HD (1280x720)

### 🖥️ Использование ресурсов
- **GPU оптимизация**: Автоматическое использование CUDA если доступна
- **Ограничение GPU памяти**: Максимум 70% от доступной памяти
- **Паузы для охлаждения**: Перерывы каждые 10 обработанных кадров
- **Асинхронная обработка**: Неблокирующие операции

### 🚦 Детекция объектов
- **Фильтрация уверенности**: Только объекты с высокой уверенностью
- **Барьеры**: Порог 0.5 (50%)
- **Съезды**: Порог 0.6 (60%)
- **Шумовые полосы**: Порог 0.6 (60%)

## 🚀 Запуск

### Требования
```bash
pip install -r requirements.txt
```

### Запуск Telegram бота
```bash
python run_bot.py
```

### Запуск в фоне
```bash
nohup python run_bot.py > bot.log 2>&1 &
```

## 📊 Возможности

- 🚦 **Дорожные знаки**: Детекция 155 классов RTSD
- 🏗️ **Инфраструктура**: Автобусы, машины, светофоры
- 🚧 **Барьеры**: Дорожные ограждения
- 🚪 **Съезды**: Выезды с трассы
- 🔊 **Шумовые полосы**: Звуковые предупреждающие полосы
- 🛣️ **Разметка**: Дорожная разметка
- 📍 **GPS трекинг**: Геолокация объектов
- 📏 **Пикетаж**: Километровые отметки

## 📋 Рекомендации по использованию

### Для предотвращения перегрева:
- Видео длительностью до 10 минут
- Размер файла до 30 MB
- Формат MP4 с кодеком H.264
- Разрешение до 1920x1080

### Оптимальные настройки:
- Короткие видео (2-5 минут)
- Среднее качество сжатия
- Стабильное освещение
- Плавное движение камеры

## 🔧 Разработка

### Добавление нового детектора:
1. Создать файл в `src/core/detectors/`
2. Реализовать функцию детекции
3. Добавить импорт в `src/telegram/bot.py`
4. Обновить пороги уверенности

### Тестирование:
```bash
python -m pytest tests/
```
