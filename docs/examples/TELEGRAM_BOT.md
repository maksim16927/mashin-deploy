# 🤖 Telegram Бот для обработки видео

## Описание

Telegram бот принимает видео от пользователя, обрабатывает его используя все модели детекции дорожной инфраструктуры, показывает прогресс обработки в процентах и отправляет обработанное видео обратно.

## Возможности

- 📹 **Прием видео** - принимает видео файлы (MP4, MOV, AVI)
- 🚦 **Детекция знаков** - 155 классов дорожных знаков (RTSD)
- 🏗️ **Инфраструктура** - автобусы, машины, светофоры (COCO)
- 🛣️ **П-образные опоры** - эвристический метод детекции
- 📏 **Дорожная разметка** - OpenCV детекция
- ⏳ **Прогресс** - показывает процент обработки в реальном времени
- 📊 **Статистика** - количество детекций каждого типа

## Установка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

Или установите только python-telegram-bot:

```bash
pip install python-telegram-bot>=20.0
```

### 2. Проверка моделей

Убедитесь, что все модели загружены:

```bash
python scripts/download_all_models.py
```

Должны быть доступны:
- `models/yolov8s_35epochs_rtsd155.pt` - дорожные знаки
- `models/yolov8n.pt` - COCO инфраструктура

## Запуск бота

### Способ 1: Через run_bot.py

```bash
python run_bot.py
```

### Способ 2: Напрямую

```bash
python src/telegram_bot.py
```

### Способ 3: В фоне (Linux/Mac)

```bash
nohup python run_bot.py > bot.log 2>&1 &
```

## Использование

1. **Найдите бота в Telegram** по токену или имени

2. **Отправьте команду `/start`** - получите приветственное сообщение

3. **Отправьте видео файл** - бот начнет обработку

4. **Следите за прогрессом** - бот будет обновлять процент обработки

5. **Получите результат** - обработанное видео будет отправлено обратно

## Настройка

### Токен бота

Токен можно задать:

1. **В коде** (не рекомендуется для продакшена):
   ```python
   BOT_TOKEN = "ваш_токен_здесь"
   ```

2. **Через переменную окружения**:
   ```bash
   export TELEGRAM_BOT_TOKEN="ваш_токен_здесь"
   python run_bot.py
   ```

3. **В .env файле** (требует python-dotenv):
   ```bash
   echo "TELEGRAM_BOT_TOKEN=ваш_токен_здесь" > .env
   ```

### Порог уверенности

В коде `src/telegram_bot.py` можно изменить:

```python
conf_threshold=0.25  # Изменить на нужное значение (0.0-1.0)
```

## Ограничения

- **Размер видео**: Telegram ограничивает размер файлов до 50MB (большие файлы отправляются как документы)
- **Форматы**: Поддерживаются MP4, MOV, AVI
- **Время обработки**: Зависит от размера видео и мощности сервера
- **Модели**: Требуют достаточно RAM (рекомендуется 8GB+)

## Устранение проблем

### Бот не запускается

```bash
# Проверьте установку python-telegram-bot
pip list | grep python-telegram-bot

# Проверьте токен
echo $TELEGRAM_BOT_TOKEN
```

### Модели не найдены

```bash
# Загрузите модели
python scripts/download_all_models.py

# Проверьте наличие моделей
ls models/*.pt
```

### Ошибки при обработке

- Проверьте формат видео (должен быть MP4, MOV, AVI)
- Убедитесь, что есть достаточно места на диске
- Проверьте логи: `tail -f bot.log`
