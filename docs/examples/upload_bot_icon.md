# 📸 Инструкция по загрузке иконки бота

## Способ 1: Через BotFather (рекомендуется)

1. Откройте Telegram и найдите бота **@BotFather**
2. Отправьте команду: `/setuserpic`
3. Выберите вашего бота из списка (или введите имя: `bottransportai`)
4. Отправьте файл `bot_icon.png` из папки проекта
5. Готово! Иконка будет применена

## Способ 2: Прямой путь к файлу

Файл иконки находится по пути:
```
/Users/shadownight/Documents/mashin/bot_icon.png
```

## Способ 3: Через скрипт (если поддерживается)

Запустите скрипт `upload_icon_to_bot.py` (если доступен)

---

**Примечание**: Изменение иконки бота доступно только через @BotFather и требует прав владельца бота.
