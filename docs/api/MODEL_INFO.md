### Использование модели

#### 1. В Python коде:
```python
from ultralytics import YOLO

# Загрузка модели
model = YOLO('models/yolov8s_35epochs_rtsd155.pt')

# Детекция на изображении
results = model.predict('image.jpg', conf=0.25)

# Детекция на видео
results = model.predict('video.mp4', conf=0.25, save=True)
```

#### 2. С интеграцией в road_detector.py:
```python
from road_detector import RoadSignDetector

# Используем готовую модель
detector = RoadSignDetector(model_path='models/yolov8s_35epochs_rtsd155.pt')

# Обработка видео
detector.process_video('video.mp4', output_path='output.mp4')
```

#### 3. Через командную строку:
```bash
# Детекция на изображении
yolo predict model=models/yolov8s_35epochs_rtsd155.pt source=image.jpg

# Детекция на видео
yolo predict model=models/yolov8s_35epochs_rtsd155.pt source=video.mp4

# С сохранением результата
yolo predict model=models/yolov8s_35epochs_rtsd155.pt source=video.mp4 save=True
```

### Рекомендации по использованию

1. **Порог уверенности (conf):** Используйте значение от 0.2 до 0.4 для баланса между точностью и полнотой
2. **Размер изображения (imgsz):** Модель обучена на 640x640, используйте это значение
3. **Batch size:** Зависит от доступной памяти GPU (8-16 для GPU, 4-8 для CPU)
4. **Постобработка:** Используйте NMS (Non-Maximum Suppression) для фильтрации дублирующихся детекций
