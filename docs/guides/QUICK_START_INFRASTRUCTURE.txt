🚀 БЫСТРЫЙ СТАРТ (3 ШАГА):
═══════════════════════════════════════════════════════════════════════════

ШАГ 1: Получите API ключ Roboflow
────────────────────────────────────────────────────────────────────────
1. Перейдите: https://app.roboflow.com/
2. Зарегистрируйтесь (можно через Google)
3. Settings → API: https://app.roboflow.com/settings/api
4. Скопируйте "Private API Key"


ШАГ 2: Загрузите датасеты
────────────────────────────────────────────────────────────────────────
python3 scripts/download_infrastructure_dataset.py --api-key "ВАШ_КЛЮЧ"

Будут загружены:
✓ datasets/Bus-Stop-Detection-1/          (автобусные остановки)
✓ datasets/road-infrastructure-predict-1/  (дороги, перекрестки)
✓ datasets/Infrastructure-Split/           (повреждения, столбы)


ШАГ 3: Обучите модель
────────────────────────────────────────────────────────────────────────
# Базовое обучение (50 эпох, ~2-3 часа на M2)
python3 scripts/train_infrastructure.py

# Или с параметрами
python3 scripts/train_infrastructure.py \
    --dataset "datasets/Bus-Stop-Detection-1/data.yaml" \
    --model yolov8s.pt \
    --epochs 100 \
    --batch 16

❓ ЧАСТЫЕ ВОПРОСЫ:
═══════════════════════════════════════════════════════════════════════════

Q: Сколько времени займет обучение?
A: На Apple M2: 2-4 часа (50-100 эпох), на NVIDIA GPU: 1-2 часа

Q: Нужно ли обучать все датасеты?
A: Можно начать с одного (например, автобусные остановки)

Q: Что если датасет не загружается?
A: Проверьте API ключ и интернет-соединение

Q: Можно ли использовать CPU?
A: Да, но обучение займет 8-12 часов

Q: Как улучшить точность?
A: Увеличьте epochs (100-200), используйте модель побольше (yolov8m/l)
