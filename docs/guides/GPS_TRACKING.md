# GPS Привязка Объектов

Система автоматически привязывает GPS координаты ко всем обнаруженным объектам на видео.

## Как это работает

1. **Извлечение GPS из видео**: Система пытается извлечь GPS координаты из метаданных видео файла
2. **Интерполяция координат**: Если GPS данные не найдены, используется интерполяция на основе:
   - Начальных координат (можно задать вручную)
   - Скорости движения (по умолчанию 50 км/ч)
   - Времени кадра
3. **Вычисление координат объектов**: Для каждого обнаруженного объекта вычисляются его GPS координаты на основе:
   - Координат центра кадра
   - Позиции объекта в кадре (bbox)
   - Параметров камеры (фокусное расстояние, высота)

## Использование

### В Telegram боте

GPS координаты автоматически добавляются в Excel отчет при обработке видео.

### В консольном скрипте

```bash
python3 process_video_console.py "videos/original/video.mp4" -o "output.mp4"
```

Координаты вычисляются автоматически и могут быть сохранены в Excel (если добавить экспорт).

## Настройка параметров

### Задание начальных координат

В коде можно задать начальные GPS координаты:

```python
from gps_tracker import GPSTracker

# Задаем начальные координаты (широта, долгота)
gps_tracker = GPSTracker(
    video_path="video.mp4",
    start_coords=(55.7558, 37.6173),  # Москва
    speed_kmh=60.0  # Скорость в км/ч
)
```

### Параметры камеры

Для более точного вычисления координат можно настроить параметры камеры:

```python
gps_tracker.set_camera_params(
    focal_length=800.0,  # Фокусное расстояние в пикселях
    altitude=5.0  # Высота камеры над землей в метрах
)
```

## Формат координат в Excel

В Excel отчете координаты сохраняются в колонках:
- **Широта** (latitude)
- **Долгота** (longitude)

Координаты в формате десятичных градусов (WGS84).

## Точность

Точность вычисления координат зависит от:
- Наличия GPS данных в видео
- Точности параметров камеры
- Высоты камеры над землей
- Расстояния до объекта

Для максимальной точности рекомендуется:
1. Использовать видео с GPS метаданными
2. Калибровать параметры камеры
3. Указывать реальную скорость движения

## Примеры использования

### Получение координат объекта

```python
from gps_tracker import GPSTracker

gps_tracker = GPSTracker("video.mp4")
gps_tracker.setup_video(cap)

# Получаем координаты объекта на кадре 100
bbox = (100, 200, 300, 400)  # x1, y1, x2, y2
coords = gps_tracker.get_object_coordinates(bbox, frame_number=100)
print(f"Координаты объекта: {coords}")  # (lat, lon)
```

### Получение координат центра кадра

```python
# Получаем координаты центра кадра
frame_coords = gps_tracker.get_frame_coordinates(frame_number=100)
print(f"Координаты кадра: {frame_coords}")
```
